생기로그 AI 사용 가이드

활동 기록 수집부터 AI 세특 초안 생성, 나이스 제출까지 — 한 곳에서 해결하세요.

선생님 · 강사 AI 세특 · 보고서 자동화 나이스 연동
💡 생기로그 AI는 사전 승인된 선생님·강사만 사용할 수 있습니다. 아직 계정이 없다면 사용 신청하기를 통해 신청해 주세요.
🌏 생기로그 AI는 고등학교 세특뿐만 아니라 중학교·학원·레슨·방과후·홈스쿨 등 수업이 있는 모든 곳에서 사용할 수 있습니다.

기능 한눈에 보기

📝
활동 기록 수집
학생이 참여 코드로 직접 제출. AI 품질 검토, 파일 첨부, 피드백, 타임라인 조회 지원.
✨
AI 세특 초안
Gemini AI가 활동 기록 기반으로 학생별 세특 초안을 자동 생성. 주차 필터·일괄 생성 지원. (학교)
📋
AI 학부모 보고서
수강생 관찰 기록 기반으로 따뜻하고 구체적인 학부모 보고서 초안을 자동 생성합니다. (학원)
📊
나이스 내보내기
500자 AI 압축, 행동특성·종합의견 초안, 나이스 엑셀 일괄 다운로드. (학교 전용)
👨‍👩‍👧‍👦
학급·학생 통합 관리
교과반·담임반 구분, 단원·출석·조 관리, 폴더 분류, 교과 교사 초대 연동.
🎒
결과물 제출
주차별 파일·링크·텍스트 결과물을 학생이 직접 첨부·제출할 수 있습니다.
🎮
실시간 퀴즈
AI 문항 자동 생성, 6자리 PIN 참여, 실시간 순위표. Kahoot 스타일 퀴즈. (Basic↑)
📋
실시간 설문
객관식·별점·단답형 등 6가지 문항 유형. AI 응답 분석·CSV 내보내기. (Basic↑)
🖼️
협업 화이트보드
실시간 조별 협업 보드. 포스트잇·도형·이미지·화살표 등 6종 오브젝트. (Basic↑)
🎙️
수업 전사 & AI 분석
수업 음성을 텍스트로 전사하고 AI가 학생별 관찰 기록을 자동 정리합니다. (Basic↑)
📚
수업 자료 에디터
마크다운 자료 작성, 슬라이드 발표 모드, 주차별 학생 공개 설정. (Basic↑)
🖼
수업 갤러리
수업 사진·영상을 학급별·주차별로 보관. WebP 자동 압축, YouTube 링크 저장.

문제 해결
🚀 처음 시작하기
계정 신청부터 첫 로그인까지
🔒 생기로그 AI는 초대 승인제로 운영됩니다. 일반 회원가입이 없으며, 관리자 승인 후 계정이 개설됩니다.

계정 신청 방법

1
사용 신청서 작성
메인 화면 하단 사용 신청하기 폼에서 이름, 이메일, 학교명, 직책을 입력하고 제출합니다.
2
검토 대기
관리자가 신청 내용을 검토합니다. 보통 1~2 영업일 내에 결과를 안내해 드립니다.
3
계정 초대 이메일 수신
승인되면 입력하신 이메일로 계정 설정 링크가 발송됩니다. 링크를 클릭해 비밀번호를 설정합니다.
4
로그인
선생님 로그인 버튼을 클릭하거나 /login으로 접속해 이메일과 비밀번호로 로그인합니다.

로그인 후 첫 화면

로그인에 성공하면 대시보드로 이동합니다. 대시보드에는 내 학급 목록, 최근 활동 기록, 초대 알림이 표시됩니다.

✅ 처음 로그인했다면 학급 만들기부터 시작하세요. 학급 없이는 다른 기능을 사용할 수 없습니다.
🏫 클래스룸
클래스를 개설하고 학생을 관리하세요
💡 클래스룸은 교과·담임 구분 없이 하나의 통합 화면으로 운영됩니다. 담임 선생님에게만 행동특성·종합의견 기능이 추가로 표시됩니다.

클래스 만들기

1
클래스룸 메뉴 접속
상단 메뉴에서 클래스룸을 클릭합니다.
2
새 클래스 만들기
우측 상단 + 새 클래스 만들기 버튼을 클릭합니다.
3
클래스 정보 입력
클래스명과 과목을 입력하고 생성합니다. 담임 선생님은 담임반 체크박스를 선택하면 행동특성 기능이 활성화됩니다.
4
학생 추가
학생을 직접 추가하거나, 생성된 입장 코드를 학생에게 공유해 자동 등록할 수 있습니다.

입장 코드 공유하기

학급을 만들면 6자리 입장 코드가 자동 생성됩니다. 학생들이 이 코드를 입력하면 해당 학급에 자동으로 등록됩니다.

대시보드 상단 아이콘 바에서 🔑 입장 코드 버튼을 클릭하면 코드가 클립보드에 복사됩니다. 복사된 코드를 학생들에게 직접 전달하세요.

QR 아이콘 버튼을 클릭하면 입장 코드가 담긴 QR코드가 생성됩니다. 교실 화면에 띄우거나 출력해서 학생들이 스캔할 수 있게 하세요.

링크 아이콘 버튼을 클릭하면 학생이 바로 입력 화면으로 이동하는 URL이 복사됩니다. 카카오톡, 문자 등으로 공유하세요.

📝 활동 기록 관리
학생의 활동 기록을 수집하고, 선생님이 직접 관찰 메모도 작성할 수 있습니다

학생 제출 기록 흐름

1
학생이 기록 제출
학생이 입장 코드로 접속해 활동 제목, 내용, 느낀 점을 작성하고 제출합니다.
2
AI 1차 검토 (선택)
선생님이 안내 지침을 설정한 경우, AI가 기록을 자동으로 검토해 good / 검토 필요 / 반려 등급을 표시합니다.
3
선생님 승인 또는 반려
제출된 기록을 확인하고 승인 또는 반려(피드백)를 선택합니다. 반려 시 학생에게 알림이 발송됩니다.
4
세특 초안 생성에 활용
승인된 기록들이 자동으로 AI 세특 초안 생성의 재료가 됩니다.

교사 직접 관찰 메모 작성

학생 제출 기록 외에, 선생님이 수업 중 직접 학생별 메모를 기록할 수 있습니다. 이 메모도 AI 세특 초안 생성에 반영됩니다.

1
학생 상세 화면 접속
클래스룸에서 학생 이름을 클릭해 학생 상세 페이지로 이동합니다.
2
교사 메모 작성
교사 메모 추가 버튼을 클릭해 수업에서 관찰한 내용을 직접 입력합니다. 주차를 선택해 주차별로 기록을 구분할 수 있습니다.
3
저장 및 세특 활용
저장된 교사 메모는 학생 기록 목록에 표시되며, 세특 초안 생성 시 자동으로 포함됩니다.
✅ 학생 제출 기록과 교사 메모를 함께 활용하면 AI 세특의 품질이 크게 향상됩니다.

세부 설정

학급 설정에서 최소 제출 글자수를 지정할 수 있습니다. 설정된 글자수 미달 시 학생이 제출 자체를 할 수 없습니다. 성의 없는 기록을 방지하는 데 효과적입니다.

자동 승인을 켜면 학생이 제출한 기록이 선생님 개입 없이 즉시 승인됩니다. 검토 부담을 줄이고 싶을 때 유용합니다.

AI 검토 안내 지침을 작성하면, 학생 제출 시 Gemini AI가 해당 기준에 따라 기록을 자동 평가합니다. 선생님에게는 AI 판단 결과가 뱃지로 표시됩니다.

⚠️ AI 검토는 참고용입니다. 최종 승인/반려는 선생님이 판단합니다.

주차별 수업 계획에서 결과 제출 필요를 체크하면, 해당 주차에 활동 기록만 제출한 학생에게 결과물도 제출하라는 알림이 표시됩니다.

✨ AI 세특 초안
활동 기록을 기반으로 AI가 세특·행특 초안을 자동 작성합니다
✅ AI 초안은 승인된 활동 기록이 1개 이상 있어야 생성됩니다. 먼저 활동 기록을 수집하고 승인해 주세요.

방법 1 — AI 초안 메뉴에서 학급·개인별 초안 생성 (권장)

상단 메뉴의 ✨ AI 초안을 이용하면 학급 전체 학생의 세특·행특 초안을 한 번에 순차 생성할 수 있습니다.

1
AI 초안 메뉴 접속
상단 메뉴에서 ✨ AI 초안을 클릭합니다. 선택한 학급이 교과반이면 AI 교과 세특 초안 생성, 담임반이면 AI 종합 생기부/행특 초안 생성 페이지가 표시됩니다.
2
대상 학급 및 학생 선택
대상 학급 선택 드롭다운에서 학급을 선택합니다. 학생 목록에서 개별로 선택하거나 전체 선택 버튼으로 모든 학생을 한 번에 선택할 수 있습니다.
3
세특 초안 생성 버튼 클릭
하단의 세특 초안 생성 버튼을 클릭합니다. Gemini AI가 학생별 활동 기록을 분석해 1명씩 순차적으로 초안을 작성합니다. 학생 수에 따라 수 분이 소요될 수 있습니다.
4
초안 검토 및 AI 교정
오른쪽 패널에서 학생별 초안을 확인합니다. 각 초안 아래 부드럽게 교정 · 내용 축소 · 전문적인 어조 버튼으로 AI 교정을 바로 요청할 수 있습니다.

방법 2 — 클래스룸에서 개별 학생 기록실로 이동해 행특 초안 생성

클래스룸에서 학생 개별 기록 페이지로 직접 이동해 행동특성 초안을 생성할 수 있습니다. 담임반에서 사용하는 경로입니다.

1
클래스룸에서 학생 사이드바 열기
클래스룸의 학생 목록에서 해당 학생 행 오른쪽의 → (화살표) 버튼을 클릭합니다. 화면 오른쪽에 학생 정보 사이드바 창이 열립니다.
2
전체 기록실 이동 버튼 클릭
사이드바 창 상단의 전체 기록실 이동 버튼을 클릭합니다. 해당 학생의 전체 기록 페이지로 이동합니다.
3
행동특성 초안 생성
전체 기록 페이지 왼쪽 패널의 행동특성 및 종합의견 초안 섹션에서 ✨ 행동특성 초안 생성 버튼을 클릭합니다. AI가 해당 학생의 전체 활동 기록을 분석해 초안을 작성합니다.
4
초안 확인 및 복사
생성된 초안을 검토하고, 나이스 붙여넣기용 복사 버튼으로 복사한 뒤 나이스 생기부에 붙여넣습니다. 내용이 마음에 들지 않으면 다시 생성하기 버튼으로 재생성할 수 있습니다.

방법 3 — 보고서 메뉴에서 세특 개별 편집

상단 메뉴의 보고서에서 학생별 세특을 직접 편집하고 AI로 보완할 수 있습니다. AI 초안 메뉴에서 생성한 초안을 여기서 불러와 사용할 수도 있습니다.

1
보고서 > 나이스 제출 준비 탭
상단 메뉴에서 보고서를 클릭한 뒤, 나이스 제출 준비 탭을 선택합니다.
2
학생 행 펼치기 후 AI 생성 또는 초안 불러오기
학생 목록에서 이름을 클릭해 행을 펼칩니다. 세특 편집 영역의 ✨ AI 생성 버튼으로 새로 생성하거나, AI 초안 불러오기 버튼으로 AI 초안 메뉴에서 만든 초안을 바로 가져올 수 있습니다.
3
초안 편집 (전체화면 가능)
텍스트 영역에서 직접 수정합니다. 편집 영역 옆 전체화면으로 보기 버튼(□ 아이콘)을 클릭하면 더 넓은 화면에서 작업할 수 있습니다.
4
500자 초과 시 압축 처리
500자를 초과하면 진행 바가 빨간색으로 표시됩니다. 문장 단위 자르기(빠름) 또는 AI로 압축(자연스러움) 중 선택해 처리합니다.
5
최종 확정 및 저장
내용이 완성되면 최종 확정 버튼을 클릭하고 저장하기로 저장합니다.

학원 강사 — 학부모 보고서 모드

학원에서는 세특 대신 학부모 성장 보고서를 생성할 수 있습니다. AI 초안 페이지에서 사용 목적을 전환하기만 하면 됩니다.

1
사용 목적 선택
AI 초안 페이지의 사용 목적 섹션에서 📋 학원 보고서를 선택합니다.
2
수강생 1명 선택 후 생성
무료 플랜은 1명씩만 생성됩니다. 수강생을 1명 선택하고 생성 버튼을 클릭합니다.
3
보고서 복사 후 전달

시작 / 일시정지 / 초기화
▶ 버튼으로 시작하고, 일시정지 후 다시 클릭하면 재개됩니다. 🔄 버튼으로 처음 시간으로 초기화합니다.

전체화면 발표 모드
⬜ 확대 버튼을 클릭하면 타이머가 전체화면으로 전환됩니다. 수업 중 TV나 프로젝터에 띄울 때 사용하세요.

조 구성 방식 선택
조 수 기준: 몇 조로 나눌지 설정 / 인원 수 기준: 조당 몇 명으로 나눌지 설정. 조장 자동 선정 여부도 선택합니다.

4

이미지로 저장하거나, 클래스에 조를 적용하면 학급 학생 목록에 조 정보가 저장됩니다. 저장된 조를 나중에 다시 불러올 수 있습니다.
💡 저장된 조 뽑기 모드: 상단 탭에서 "저장된 조"를 선택하면 클래스에 저장된 조 배정을 다시 사용할 수 있습니다.

문제 진행 및 결과 확인
참가자가 모이면 퀴즈 시작을 눌러 진행합니다. 각 문제 후 정답과 실시간 순위가 표시됩니다. 마지막엔 최종 순위 화면이 나옵니다.

🖼️ 협업 화이트보드 — Basic 이상 (Basic: 최대 3보드 / Pro: 무제한)

1

드롭다운에서 전사할 클래스를 선택합니다. 선택한 클래스의 학생 이름이 AI 분석의 기준이 됩니다.

3
녹음 중지 및 AI 분석
■ 중지 버튼으로 녹음을 끝냅니다. 이후 ✨ AI 분석을 클릭하면 Gemini AI가 전사 내용을 3가지 관점으로 분석합니다.
4
분석 결과 활용
학생별 관찰: 언급된 학생의 참여도와 메모 초안 / 수업 평가: 구성·명확성·참여·피드백 점수 / 자기 성찰: 발화 패턴·개선점. 관찰 초안은 바로 학생 기록에 저장할 수 있습니다.
🔑 Basic 플랜 전사 설정: 설정(⚙️) 메뉴 → Groq API 키 입력란에 본인의 Groq API 키를 등록하면 전사 기능을 사용할 수 있습니다. Groq API 키는 console.groq.com에서 무료로 발급받을 수 있습니다.
⚠️ 전사 기능은 Chrome 브라우저에서 가장 잘 작동합니다. Safari나 Firefox에서는 실시간 전사가 지원되지 않을 수 있습니다.
💡 기록 보기 탭에서 이전에 진행한 수업 전사 기록을 다시 확인하고 재분석할 수 있습니다.

학급 및 학생

클래스룸은 단일 화면으로 통합되어 있습니다. 클래스 생성 시 담임반 옵션을 선택하면 행동특성·종합의견 기능이 추가로 활성화됩니다. 한 계정으로 여러 클래스를 만들 수 있습니다.
교과 대시보드 상단 아이콘 바에서 🔑 입장 코드 버튼을 클릭하면 코드를 바로 복사할 수 있습니다. 학생에게 다시 공유하세요.
학생 목록에서 해당 학생의 이름을 클릭하면 인라인 편집이 가능합니다. 수정 후 Enter를 누르면 저장됩니다.

활동 기록 및 세특

네. ✨ AI 초안 생성 버튼을 다시 클릭하면 새로운 초안이 생성됩니다. 이전 버전으로 되돌리려면 되돌리기 버튼을 사용하세요. (세션 내에서만 가능)
데이터 내보내기 > 나이스 제출 준비 탭에서 학생 행을 펼쳤을 때 500자를 초과하면 AI로 압축 버튼이 나타납니다. 클릭하면 내용의 핵심을 유지하면서 500자 이하로 자동 압축합니다. 빠르게 자르려면 문장 단위 자르기 버튼을 사용하세요.
AI 세특의 품질은 활동 기록의 양과 질에 따라 달라집니다. 학생이 더 많은 기록을 제출할 수 있도록 최소 글자수 설정과 주차별 학습 계획 등록을 권장합니다.
다운로드된 엑셀 파일을 열면 학생별 세특이 정리되어 있습니다. 나이스 워크스테이션에서 세특 입력 시 해당 내용을 복사해 붙여넣으면 됩니다.
