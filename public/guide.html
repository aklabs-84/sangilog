생기로그 AI 사용 가이드

관찰기록 수집부터 AI 세특 초안 생성, 나이스 제출까지 — 한 곳에서 해결하세요.

선생님 · 강사 AI 세특 · 보고서 자동화 나이스 연동
💡 생기로그 AI는 사전 승인된 선생님·강사만 사용할 수 있습니다. 아직 계정이 없다면 사용 신청하기를 통해 신청해 주세요.

핵심 기능 한눈에 보기

📝
관찰기록 수집
학생·수강생이 참여 코드로 직접 제출. 선생님·강사는 승인만 합니다.
✨
AI 세특 초안
Gemini AI가 관찰기록을 분석해 세특 초안을 생성합니다. (학교)
📋
AI 학부모 보고서
관찰기록을 기반으로 학부모 전달용 성장 보고서 초안을 자동 생성합니다. (학원)
📊
나이스 내보내기
500자 맞춤 편집 후 나이스 엑셀로 한 번에 다운로드합니다. (학교)
👨‍👩‍👧‍👦

담임반·교과반·수강반을 구분해 학생과 수업을 체계적으로 관리합니다.
🎒
결과물 제출

계정 신청부터 첫 로그인까지
🔒 생기로그 AI는 초대 승인제로 운영됩니다. 일반 회원가입이 없으며, 관리자 승인 후 계정이 개설됩니다.

계정 신청 방법

1
사용 신청서 작성
메인 화면 하단 사용 신청하기 폼에서 이름, 이메일, 학교명, 직책을 입력하고 제출합니다.
2
검토 대기
관리자가 신청 내용을 검토합니다. 보통 1~2 영업일 내에 결과를 안내해 드립니다.
3
계정 초대 이메일 수신
승인되면 입력하신 이메일로 계정 설정 링크가 발송됩니다. 링크를 클릭해 비밀번호를 설정합니다.
4
로그인
선생님 로그인 버튼을 클릭하거나 /login으로 접속해 이메일과 비밀번호로 로그인합니다.

로그인 후 첫 화면

로그인에 성공하면 대시보드로 이동합니다. 대시보드에는 내 학급 목록, 최근 관찰기록, 초대 알림이 표시됩니다.

✅ 처음 로그인했다면 학급 만들기부터 시작하세요. 학급 없이는 다른 기능을 사용할 수 없습니다.
🏫 학급 만들기
교과반과 담임반, 목적에 맞게 개설하세요

학급 유형 선택

유형특징주요 기능
교과반특정 과목을 가르치는 수업세특 작성, 관찰기록, 결과물 제출
담임반학급 전체를 담당하는 담임행동특성·종합의견, 교과 교사 연동, 종합 관리

학급 만들기

1
클래스룸 메뉴 접속
왼쪽 사이드바에서 클래스룸을 클릭합니다.
2
새 학급 만들기 클릭
우측 상단 + 새 학급 만들기 버튼을 클릭합니다.
3
학급 정보 입력
학급명, 유형(교과반/담임반), 과목을 입력하고 생성합니다.
4
학생 추가
학생을 직접 추가하거나, 생성된 입장 코드를 학생에게 공유해 자동 등록할 수 있습니다.

입장 코드 공유하기

학급을 만들면 6자리 입장 코드가 자동 생성됩니다. 학생들이 이 코드를 입력하면 해당 학급에 자동으로 등록됩니다.

대시보드 상단 아이콘 바에서 🔑 입장 코드 버튼을 클릭하면 코드가 클립보드에 복사됩니다. 복사된 코드를 학생들에게 직접 전달하세요.

QR 아이콘 버튼을 클릭하면 입장 코드가 담긴 QR코드가 생성됩니다. 교실 화면에 띄우거나 출력해서 학생들이 스캔할 수 있게 하세요.

링크 아이콘 버튼을 클릭하면 학생이 바로 입력 화면으로 이동하는 URL이 복사됩니다. 카카오톡, 문자 등으로 공유하세요.

담임반 — 교과 선생님 연동

담임반에서 여러 교과 선생님의 세특을 통합 관리할 수 있습니다.

1
교과 선생님 초대
담임반 대시보드 상단 👤 교사 초대 버튼을 클릭합니다.
2
공유 링크 전달
생성된 링크를 교과 선생님에게 공유합니다. 선생님이 링크를 수락하면 연동이 완료됩니다.
3
연동 확인
연동된 교과반의 세특이 담임반 학생별 화면에 통합되어 표시됩니다.
📝 관찰기록 관리
학생의 활동 기록을 수집하고 관리합니다

관찰기록 흐름

1
학생이 기록 제출
학생이 입장 코드로 접속해 활동 제목, 내용, 느낀 점을 작성하고 제출합니다.
2
AI 1차 검토 (선택)
선생님이 안내 지침을 설정한 경우, AI가 기록을 자동으로 검토해 good / 검토 필요 / 반려 등급을 표시합니다.
3
선생님 승인 또는 반려
제출된 기록을 확인하고 승인 또는 반려(피드백)를 선택합니다. 반려 시 학생에게 알림이 발송됩니다.
4
세특 초안 생성에 활용
승인된 기록들이 자동으로 AI 세특 초안 생성의 재료가 됩니다.

세부 설정

학급 설정에서 최소 제출 글자수를 지정할 수 있습니다. 설정된 글자수 미달 시 학생이 제출 자체를 할 수 없습니다. 성의 없는 기록을 방지하는 데 효과적입니다.

자동 승인을 켜면 학생이 제출한 기록이 선생님 개입 없이 즉시 승인됩니다. 검토 부담을 줄이고 싶을 때 유용합니다.

AI 검토 안내 지침을 작성하면, 학생 제출 시 Gemini AI가 해당 기준에 따라 기록을 자동 평가합니다. 선생님에게는 AI 판단 결과가 뱃지로 표시됩니다.

⚠️ AI 검토는 참고용입니다. 최종 승인/반려는 선생님이 판단합니다.

주차별 수업 계획에서 결과 제출 필요를 체크하면, 해당 주차에 관찰기록만 제출한 학생에게 결과물도 제출하라는 알림이 표시됩니다.

✨ AI 세특 초안
승인된 관찰기록을 기반으로 AI가 세특을 자동 작성합니다
✅ AI 세특 초안은 승인된 관찰기록이 1개 이상 있어야 생성됩니다. 먼저 관찰기록을 수집하고 승인해 주세요.

세특 초안 생성하기

1
학생 상세 페이지 접속
학급 대시보드에서 학생 이름을 클릭해 학생 뷰로 이동합니다.
2
AI 초안 생성 버튼 클릭
세특 편집 영역 우측의 ✨ AI 초안 생성 버튼을 클릭합니다. Gemini AI가 승인된 기록을 분석해 초안을 작성합니다.
3
초안 검토 및 편집
생성된 초안을 직접 수정합니다. 전체화면 편집 모드(□ 확대 버튼)에서 더 편하게 작업할 수 있습니다.
4
500자 맞춤 처리
나이스 제출 기준인 500자를 초과하면 AI 압축 버튼으로 자동으로 줄여줍니다. 수동 편집도 가능합니다.

일괄 생성 (AI 어시스턴트)

개별 학생이 아닌 학급 전체의 세특을 한 번에 생성하려면 사이드바의 AI 어시스턴트를 이용하세요.

1
AI 어시스턴트 메뉴 접속
왼쪽 사이드바에서 AI 어시스턴트를 클릭합니다.
2
학급 선택 후 일괄 생성
대상 학급을 선택하고 전체 세특 초안 생성을 클릭합니다. 학생 수에 따라 수 분이 소요될 수 있습니다.

행동특성 및 종합의견 (담임반 전용)

담임반에서는 학생별 행동특성·종합의견 초안도 AI가 자동 생성합니다. 학생 상세 페이지 하단에서 확인하고 복사할 수 있습니다.

학원 강사 — 학부모 보고서 모드

학원에서는 세특 대신 학부모 성장 보고서를 생성할 수 있습니다. AI 어시스턴트 화면에서 사용 목적을 전환하기만 하면 됩니다.

1
사용 목적 선택
AI 어시스턴트 화면의 사용 목적 섹션에서 📋 학원 보고서를 선택합니다.
2
수강생 1명 선택 후 생성
무료 플랜은 1명씩만 생성됩니다. 수강생을 1명 선택하고 생성 버튼을 클릭합니다.
3
보고서 복사 후 전달
생성된 보고서를 복사해 학부모에게 문자·카카오톡으로 전달합니다.
💡 되돌리기 기능: 세션 내에서는 이전 세특 내용으로 되돌릴 수 있습니다. 페이지를 새로고침하면 되돌리기 기록이 초기화됩니다.
📊 나이스 내보내기
완성된 세특을 나이스 엑셀 형식으로 다운로드합니다

나이스 워크스테이션 사용하기

1
데이터 내보내기 메뉴 접속
사이드바에서 데이터 내보내기를 클릭하고 나이스 워크스테이션 탭을 선택합니다.
2
학급 및 학생 선택
내보낼 학급을 선택합니다. 학생 목록이 표시됩니다.
3
세특 내용 최종 확인
학생별 세특 내용을 확인하고 필요시 마지막 수정을 합니다. 500자 초과 학생은 빨간색으로 표시됩니다.
4
엑셀 다운로드
나이스 엑셀 다운로드 버튼을 클릭합니다. 나이스 워크스테이션에서 바로 붙여넣기 할 수 있는 형식으로 저장됩니다.
⚠️ 나이스 세특 입력 기준 500자를 초과하면 제출이 거부됩니다. 다운로드 전에 모든 학생의 세특이 500자 이하인지 확인하세요.

🎰

수업 자료실

📋
퀴즈 게임

📢
건의사항 관리

학급 공유 링크
로그인 없이 학급 현황을 볼 수 있는 공개 링크를 생성합니다
🎒 학생 참여 방법
학생 입장에서의 이용 방법을 안내합니다
💡 학생은 별도 회원가입이 필요 없습니다. 선생님께 받은 입장 코드만 있으면 됩니다.

학급 입장하기

1
생기로그 AI 접속
메인 화면(생기로그 AI) 접속 후 수업 참여 버튼을 클릭합니다. 또는 선생님이 공유한 링크로 바로 이동합니다.
2
입장 코드 입력
선생님께 받은 6자리 입장 코드를 입력합니다.
3
이름 확인
학생 목록에서 자신의 이름을 선택합니다.

학원에서 시작하는 방법

1
수강반 만들기
클래스룸에서 + 새 학급 만들기를 클릭해 수강반을 개설합니다. 이름 예시: "중등 수학 A반", "초등 영어 3시반"
2
수강생 등록 및 코드 공유
수강생을 직접 추가하거나 입장 코드를 수강생에게 공유합니다. 수강생은 회원가입 없이 코드만으로 접속합니다.
3
수업 관찰기록 쌓기
수강생이 수업 후 느낀 점·활동 내용을 직접 제출하거나, 강사가 직접 수강생별 메모를 입력합니다.
4
AI 학부모 보고서 생성
AI 어시스턴트 메뉴에서 수강반을 선택하고 사용 목적 → 📋 학원 보고서를 선택한 뒤 생성합니다.
5
학부모에게 전달
생성된 보고서 초안을 검토·수정 후 복사해 문자, 카카오톡, 알림장 앱 등에 붙여넣어 전달합니다.

학부모 보고서 작성 팁

✅ 관찰기록이 많을수록 보고서 품질이 높아집니다. 매 수업 후 수강생이 짧게라도 기록을 제출하도록 습관을 만들어 주세요.
⚠️ AI가 생성한 보고서는 초안입니다. 학부모에게 전달하기 전 반드시 내용을 검토하고 수강생에게 맞게 수정해 주세요.

자주 묻는 질문 (학원)

네. 학교 선생님과 동일하게 무료(Free) 플랜으로 시작할 수 있습니다. 수강반 2개, AI 보고서 하루 10회까지 무료입니다. 더 많이 사용하려면 Pro 플랜으로 업그레이드하세요.
나이스는 학교 교사 전용 시스템이므로 학원에서는 해당 기능을 사용할 필요가 없습니다. 대신 AI가 생성한 학부모 보고서를 복사해 직접 전달하는 방식을 사용하세요.
현재 PDF 직접 내보내기 기능은 없습니다. 브라우저의 인쇄 기능(Ctrl+P)을 이용해 PDF로 저장하거나, 텍스트를 복사해 워드·한글 문서로 만들어 사용하세요.
❓ 자주 묻는 질문
선생님들이 자주 질문하시는 내용을 모았습니다

계정 및 접속

생기로그 AI는 열린 회원가입이 없습니다. 메인 페이지에서 사용 신청하기를 통해 신청하면 관리자 검토 후 계정 초대 이메일을 발송합니다.
로그인 화면 하단 도움이 필요하신가요?를 클릭해 비밀번호 재설정 이메일을 받으세요. 또는 관리자에게 문의해 주세요.
네. 모바일 브라우저에서도 사용 가능합니다. 단, 세특 작성 및 엑셀 다운로드는 PC 환경에서 더 편리합니다.

학급 및 학생

네. 한 계정으로 담임반과 교과반을 모두 여러 개 만들 수 있습니다. 왼쪽 상단 학급 선택기로 전환합니다.
교과 대시보드 상단 아이콘 바에서 🔑 입장 코드 버튼을 클릭하면 코드를 바로 복사할 수 있습니다. 학생에게 다시 공유하세요.
학생 목록에서 해당 학생의 이름을 클릭하면 인라인 편집이 가능합니다. 수정 후 Enter를 누르면 저장됩니다.

관찰기록 및 세특

네. ✨ AI 초안 생성 버튼을 다시 클릭하면 새로운 초안이 생성됩니다. 이전 버전으로 되돌리려면 되돌리기 버튼을 사용하세요. (세션 내에서만 가능)
세특 편집창에서 AI 압축 버튼을 클릭하면 내용의 핵심을 유지하면서 500자 이하로 자동 압축합니다.
AI 세특의 품질은 관찰기록의 양과 질에 따라 달라집니다. 학생이 더 많은 기록을 제출할 수 있도록 최소 글자수 설정과 주차별 학습 계획 등록을 권장합니다.
다운로드된 엑셀 파일을 열면 학생별 세특이 정리되어 있습니다. 나이스 워크스테이션에서 세특 입력 시 해당 내용을 복사해 붙여넣으면 됩니다.
🔧 문제 해결
자주 발생하는 오류와 해결 방법

로그인 문제

브라우저 캐시를 지우고 다시 시도해 보세요. 또는 시크릿 창에서 접속해 보세요. 문제가 계속되면 관리자에게 문의해 주세요.
계정 초대 이메일에서 설정한 비밀번호를 확인하세요. 기억이 나지 않으면 로그인 화면에서 도움이 필요하신가요?를 통해 재설정하세요.

기능 오류

원인 1: 승인된 관찰기록이 없는 경우입니다. 먼저 학생 기록을 승인하세요.
원인 2: 일시적인 AI 서버 오류일 수 있습니다. 30초 후 다시 시도해 보세요.
입장 코드를 다시 확인해 주세요. 대소문자가 구분됩니다. 교과 대시보드 상단에서 올바른 코드를 복사해 학생에게 다시 공유하세요.
브라우저 팝업 차단 설정을 확인하세요. 다운로드가 차단된 경우 주소창 우측에 팝업 차단 아이콘이 표시됩니다. 허용 후 다시 시도하세요.
브라우저를 새로고침(F5 또는 Ctrl+R)하거나, 캐시를 지우고(Ctrl+Shift+Delete) 다시 접속해 보세요.
📧 위 방법으로 해결되지 않으면 관리자에게 문의해 주세요. 문의 시 오류 메시지와 스크린샷을 함께 보내주시면 빠른 도움이 가능합니다.
